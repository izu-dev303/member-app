# Flask 会員管理アプリ

## アプリ概要
Flaskで作成した会員情報のCRUD管理アプリです。
Python学習10日目に独学で作成しました。

## 機能一覧
- 会員の登録・編集・削除（CRUD全機能）
- 名前での検索機能
- CSVダウンロード機能
- Bootstrapによるレスポンシブデザイン

## 使用技術
- Python 3.x
- Flask
- HTML / CSS
- Bootstrap 5
- CSV（データ保存）

## 起動方法
```bash
pip install flask
python app.py
```
ブラウザで `http://127.0.0.1:5000` を開く
